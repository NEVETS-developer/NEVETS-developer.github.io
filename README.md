# Steven Moleshi - Portfolio Website

This is the portfolio website for **Steven Moleshi**, showcasing skills, projects, and contact information.

## Features

- **Hero Section**: Introduction with a professional design.
- **About Section**: Brief description of Steven's journey and expertise.
- **Services Section**: Highlights programming languages and skills.
- **Projects Section**: Displays completed projects with links to view them.
- **Contact Section**: Provides contact details and social media links.

- **CSS3**
- **JavaScript**
